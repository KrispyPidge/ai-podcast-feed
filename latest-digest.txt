Good morning! Here’s your daily A.I. digest for Saturday, April eleventh, 2026.

Starting with the top story. Anthropic made waves this week by announcing Project Glasswing, and with it the reveal that their latest model, Claude Mythos Preview, is too capable to release to the public. Specifically, its cybersecurity abilities are in a league of their own. Mythos has already discovered thousands of high-severity vulnerabilities across every major operating system and web browser. In one case, it chained together four, vulnerabilities to write a browser exploit that escaped both the renderer and Oh Ess sandboxes. On OpenBSD, it found a bug that had been sitting there for twenty-seven years.

To put this in perspective, Claude Opus four, point six had a near-zero success rate at autonomous exploit development. Mythos Preview developed working Firefox exploits one hundred and eighty-one times out of several hundred attempts. That is an enormous jump.

Anthropic is backing the project with one hundred million dollars in usage credits and four, million dollars in direct donations to open-source security organizations. Partners include Amazon Web Services, Apple, Microsoft, Google, and the Linux Foundation. And Anthropic isn’t alone here, OpenAI has reportedly built a similar cybersecurity-focused model and plans to release it to a small set of partners as well.

Linux kernel maintainer Greg Kroah-Hartman summed up the shift well. He said that months ago, A.I.-generated security reports were obvious slop. Now, they are real, and they are good. The curl lead developer Daniel Stenberg says he is spending hours per day dealing with the resulting flood of legitimate vulnerability reports. For anyone running game servers or managing infrastructure, the takeaway is clear, A.I.-discovered vulnerabilities are about to accelerate the patching treadmill for everyone.

Next up, Ell Ell Ems and coding tools. Meta released Muse Spark, their first frontier model since Llama four, almost exactly a year ago. It comes with two modes on the meta dot A.I. chat, Instant and Thinking, with a deeper Contemplating mode promised for the future. Meta claims Muse Spark reaches equivalent capabilities with over ten times less compute than Llama four, Maverick. Simon Willison dug into the meta dot A.I. interface and found sixteen tools wired in, including code execution, image generation, visual grounding, sub-agent spawning, and cross-platform content search across Instagram, Threads, and Facebook. Meta has hinted at open-sourcing future versions, so watch that space.

But developer sentiment tells a different story. And some quick hits. The Axios JavaScript library was hit by a supply chain attack this week, and the method was disturbingly sophisticated. Every open-source maintainer should know about this attack pattern.

And finally, one thing to try today. Set up a Notion Custom Agent for your recurring community management tasks. The Rundown featured a guide on building Notion agents that run on a schedule. Clone the pattern for different recurring jobs, content review, community metrics, stream schedule planning. Same two databases, different agent, every run traceable in one place.
